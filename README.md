大涼國國會官網　README

> 以下內容以繁體字中文書寫，並採用傳統直排（自右而左）之書面語氣。 實際顯示為橫排文字，惟行文、標點與結構皆依直排文書之閱讀習慣設計。



———

一、本站性質

大涼國國會官網者， 乃聖世眾生共政之最高公共資訊樞紐也。

此站非一黨一派之私產， 亦非行政機關之附屬門戶， 而為——

> 全體眾生共議、共書、共知之國會正站。



凡關乎： 國體、政體、制度、法理、議事、公告、檔案、史料者， 皆以此站為最高公開來源。

———

二、國會定位

大涼國國會， 非代議眾生， 而為眾生自議之制度化呈現。

故其官網職能在於：

記錄眾生共識

公示制度文本

彙整文明文獻

留存歷史決議

提供開放查閱


本站不以「管理」為核心， 而以「流通」為本義。

———

三、內容結構（約定）

本站內容大體分為數域：

（一）國會公告

國會決議

公告通告

制度啟示

臨時聲明


（二）根本文獻

《妙上百觀經》

國體原理文本

聖世憲制文獻

自然法相關闡述


（三）議事存檔

歷次議程

討論紀錄

形成文本

眾生意見彙編


（四）文明與歷史

明華文明史綱

大涼國成立文獻

相關文明比較材料


———

四、語言與文字規範

全站統一使用：繁體中文

不使用簡化字

不強制白話或文言

視內容性質，自然分流


經、典、制、憲，多用典雅書面語； 公告、說明、導引，從眾生可讀性。

———

五、技術原則

本站遵循以下原則：

重內容，輕炫技

可長期保存

結構清晰

去中心化依賴


不以商業平台為核心， 不引入不必要之追蹤、演算法或娛樂化模組。

———

六、關於參與

大涼國國會官網， 不設「旁觀者」。

凡眾生， 皆為歷史之書寫者。

然一切參與， 須遵循：

> 理性、克制、尊重自然法、尊重文明整體。



———

七、聲明

本站所載之一切文本， 皆屬聖世文明公共遺產。

不以個人崇拜為目的， 不以情緒動員為手段， 不以神秘化、偶像化為路徑。

若讀之而有所思， 已是其用。

———
